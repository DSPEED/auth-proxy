Auth-proxy

Check incoming frontend request against your API to see if a user is logged in

Download this project as a .zip file Download this project as a tar.gz file

Purpose of this module

This Node.js module is made for a scenario where you use Node.js and separate frontend from the API. For incoming requests we need to know if the user is logged in and what rights (s)he has.

Where it runs

The module runs on your frontend server and enables you to specify a URL that will be called. You can pass along a function that will be called to ensure that the user is valid (i.e. check the response from the API).
